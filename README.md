# Microsoft Full-Stack Developer — Coursework Repository

## 📌 About

This repository is a structured learning portfolio for the [Microsoft Full-Stack Developer Professional](https://www.coursera.org/professional-certificates/microsoft-full-stack-developer?utm_medium=sem&utm_source=gg&utm_campaign=b2c_latam_x_multi_ftcof_career-academy_cx_dr_bau_gg_pmax_gc_s1-v2_en_m_hyb_24-04_nonNRL-within-14d&campaignid=21239613742&adgroupid=&device=c&keyword=&matchtype=&network=x&devicemodel=&creativeid=&assetgroupid=6501905150&targetid=&extensionid=&placement=&gad_source=1&gad_campaignid=21320925518&gbraid=0AAAAADdKX6apNDZuqeuqTeNHo6LWryDrU#outcomes) Certificate program.

It contains course notes, coding activities, practice exercises, and projects completed while learning full-stack development with C#, .NET, frontend technologies, databases, APIs, security, and DevOps practices.

The purpose of this repository is to document my progress, organize my coursework, and demonstrate the practical skills I develop throughout the program.

Main objectives:

- Learn and apply C# and .NET fundamentals
- Build frontend interfaces with HTML, CSS, JavaScript, Blazor, and React
- Develop backend services and REST APIs with ASP.NET Core
- Work with databases using SQL and Entity Framework Core
- Practice authentication, authorization, performance optimization, and deployment
- Use Git and GitHub to manage and document my learning progress
- Complete a full-stack capstone project that combines frontend, backend, database, and deployment workflows

## 🗂 Repository Structure

This repository is organized by course modules.   
Each folder contains notes, activities, exercises, and projects related to a specific topic in the Microsoft Full-Stack Developer learning path.

- `01-Foundations_of_Coding_Full-Stack/` — full-stack concepts, algorithms, flowcharts, control structures, and C# basics.
- `02-Introduction_to_Programming_with_CSharp/` — C# fundamentals, .NET tooling, OOP, and asynchronous programming.
- `03-Inroduction_to_Web_Development/` — HTML, CSS, JavaScript, Git, and GitHub.
- `04-Blazor_for_Fron-End_Development/` — Blazor and front-end development with .NET.
- `05-Back-End_Development_with_DotNet/` — ASP.NET Core, APIs, and backend services.
- `06-Database_Integration_and_Management/` — databases, SQL, and Entity Framework Core.
- `07-Full-Stack_Integration/` — connecting frontend, backend, and database layers.
- `08-Security_and_Authentication/` — authentication, authorization, and secure application practices.
- `09-Perfomance_Optimization_and_Scalability/` — performance, scalability, and optimization techniques.
- `10-Data_Structures_and_Algorithms/` — data structures, algorithms, and problem solving.
- `11-Deployment_and_DevOps/` — deployment, CI/CD, GitHub Actions, and DevOps basics.
- `12-Full-Stack_Developer_Capstone_Project/` — final full-stack capstone project.
